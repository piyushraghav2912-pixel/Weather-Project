# Power BI Dashboard Project
This is an end-to-end Power BI project created to understand the complete data analytics workflow.  
The project covers data import, data cleaning, data modeling, and dashboard creation using Microsoft Power BI.

## Tools Used
- Microsoft Power BI  
- Power Query  
- DAX  

## Key Learnings
- Data transformation using Power Query  
- Creating relationships and measures  
- Designing interactive dashboards
